# 機能一覧

## 1. 認証機能

### 1.1 ユーザー登録
- メールアドレスとパスワードによる登録
- メール認証（オプション）
- 利用規約への同意

### 1.2 ログイン・ログアウト
- メールアドレス・パスワードによるログイン
- 「ログイン状態を保持する」機能
- ログアウト機能

### 1.3 パスワードリセット
- パスワード忘れ時のリセット機能
- メール送信によるリセットリンク発行

### 1.4 ゲストアクセス
- 認証不要での基本ページ閲覧
- ゲスト予約機能

---

## 2. 店舗管理機能

### 2.1 店舗情報管理（店舗店長・システム管理者）
- 店舗名、住所、電話番号の登録・編集
- 店舗説明文の登録・編集
- 店舗画像のアップロード・削除
- 営業時間の設定
- 休業日の設定
- 店舗の公開・非公開設定

### 2.2 店舗情報閲覧（全ユーザー・ゲスト）
- 店舗一覧の表示
- 店舗詳細ページの表示
- 店舗の検索・フィルタリング
- 地図表示（オプション）

---

## 3. メニュー管理機能

### 3.1 メニュー管理（店舗店長・システム管理者）
- ヘアメニュー名、説明、料金の登録・編集・削除
  - カット（ショート、ミディアム、ロングなど）
  - カラー（全体カラー、部分カラー、ハイライトなど）
  - パーマ（ストレートパーマ、縮毛矯正、デジタルパーマなど）
  - トリートメント
  - ヘッドスパ
  - ヘアセット
  - その他ヘアメニュー
- メニュー画像のアップロード・削除
- 所要時間の設定（カット30分、カラー90分など）
- カテゴリーの設定（カット、カラー、パーマ、トリートメントなど）
- メニューの公開・非公開設定
- スタイリストとの紐付け

### 3.2 メニュー閲覧（全ユーザー・ゲスト）
- メニュー一覧の表示
- メニュー詳細の表示
- メニューの検索・フィルタリング

---

## 4. 予約管理機能

### 4.1 予約作成（一般ユーザー・ゲスト）
- 予約フォームでの予約作成
- 希望日時の選択
- メニューの選択（カット、カラー、パーマなど）
- 複数メニューの同時予約（カット+カラーなど）
- スタイリストの選択（オプション）
- 連絡先情報の入力
- 備考欄の入力（希望スタイル、カラー希望など）
- 予約確認メールの送信

### 4.2 予約管理（一般ユーザー）
- マイページでの予約一覧表示
- 予約詳細の確認
- 予約の変更（キャンセルポリシーに基づく）
- 予約のキャンセル
- 予約履歴の閲覧

### 4.3 予約管理（店舗店長）
- 予約一覧の表示（カレンダー・リスト表示）
- 予約詳細の確認
- 予約の承認・却下
- 予約の編集（日時、メニュー、スタイリストの変更）
- 複数メニューの予約管理（カット+カラーの時間調整など）
- 予約のキャンセル
- 予約状況の統計表示
- メニュー別の予約状況分析

### 4.4 予約閲覧（店舗スタイリスト）
- 予約一覧の閲覧
- 予約詳細の閲覧
- 自分の担当予約のフィルタリング
- 自分のスケジュール確認

### 4.5 予約通知
- 予約確定時のメール通知
- 予約リマインダー（前日・当日）
- 予約変更・キャンセル時の通知

## 14. カウンセリングシート機能

### 14.1 カウンセリング情報入力（一般ユーザー・ゲスト）

- パーマ希望の記入（ストレートパーマ、縮毛矯正など）

- 過去の施術歴の記入（前回のカット、カラー、パーマなど）
- カラー剤・パーマ剤のアレルギー情報の記入

### 14.2 カウンセリング記録管理（店舗スタイリスト・店舗店長）
- カウンセリング時の追加記録
- 施術内容の記録（カットの長さ、カラーの色番、パーマの種類など）
- 使用したカラー剤・パーマ剤の記録
- 施術時間の記録
- 次回の提案内容の記録（次回のカット周期、カラー提案など）
- 過去のカウンセリング履歴の閲覧
- 顧客の好みのスタイルの記録

### 14.3 カウンセリング履歴閲覧（一般ユーザー）
- 自分のカウンセリング履歴の閲覧
- 過去の要望・提案内容の確認
