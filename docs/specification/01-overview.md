# プロジェクト概要

## システム名
美容室サロン予約システム

## 目的
美容室（理髪店）の予約管理、スタイリスト管理、シフト管理を統合的に行うためのWebアプリケーションシステム。カット、カラー、パーマなどのヘアサロン業務に特化した機能を提供します。

## 主な機能
- **店舗管理**: 店舗情報の管理・公開、営業時間・休業日の設定
- **メニュー管理**: ヘアメニュー（カット、カラー、パーマ、トリートメントなど）の管理、料金・所要時間の設定
- **予約管理**: 予約の作成・管理・キャンセル、キャンセル待ち機能
- **顧客管理**: 顧客カード、来店履歴、髪質・希望スタイル・カラー希望の記録、カラー剤アレルギー情報管理
- **ポイント・クーポン**: ポイント付与システム、クーポン発行、会員ランク制度
- **レビュー・評価**: 施術後のレビュー投稿・閲覧、店舗・スタッフへの評価
- **カウンセリング**: カウンセリングシートによる情報管理（髪質、希望スタイル、カラー希望など）、施術履歴の記録
- **施術写真**: カット前後・カラー前後の写真管理・顧客への共有
- **スタイリスト管理**: スタイリスト情報、スキル（カット、カラー、パーマなど）、実績、評価の管理
- **シフト管理**: シフトの作成・管理、カレンダー表示
- **在庫管理**: カラー剤・パーマ剤・シャンプー・トリートメント剤などの在庫管理、材料費の記録
- **売上管理**: 売上集計・分析、メニュー別・スタッフ別のレポート
- **通知機能**: 予約リマインダー、ポイント付与、クーポン通知など
- **ユーザー管理**: 権限管理、アクティビティログによる操作履歴の記録

## 対象ユーザー
1. **システム管理者（Admin）**: システム全体の管理を行う
2. **店舗店長**: 店舗、メニュー、予約、シフトの管理を行う
3. **店舗スタイリスト**: スタイリスト情報、予約情報、シフト情報の閲覧を行う
4. **一般ユーザー**: 予約の作成・管理、マイページの利用
5. **ゲスト**: 店舗情報などの基本ページの閲覧

## システムの特徴
- マルチテナント対応（複数店舗の管理が可能）
- ロールベースのアクセス制御（RBAC）
- リアルタイムな予約状況の確認
- 操作履歴の完全な記録
- レスポンシブデザイン対応
